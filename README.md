✨ Core Capabilities & Features
🔹 Self-Automated Invoice Engine (invoice_engine.php)
No manual billing management needed. The background engine acts as an independent cron handler, running once a day to check upcoming loan cycles, calculate combined interest totals, and automatically issue new statements 5 days before their due date.

🔹 Dynamic Multi-Tier Limits Mapping
Control borrowing limits cleanly through the administrative configuration block. Set risk constraints across multiple levels (Level 1: ₱2,000 Cap @ 30%, Level 2: ₱5,000 Cap @ 25%, Level 3: ₱10,000 Cap @ 20%) to keep capital distributions fully safe.

🔹 Closed-Loop GCash Payment Reconciliation
Clients submit unique reference numbers along with visual screenshot images directly to the payment system. Administrators can look up the unverified entries inside the queue layout and instantly sign off on or decline payments with specific review logs.

🔹 Referral Agent Index Directory
Track external agents inside an integrated master database ledger. This setup allows you to trace applications back to specific referrers, verify contact parameters, and prevent unauthorized client signups.

🔹 Telegram Webhook Alert Pipelining
Connect the platform directly to your private staff groups. The built-in framework fires immediate alerts to your chat channel the second a user submits an identity document or applies for credit.

🛡️ Enterprise Security Out-of-the-Box
SQL Injection Guard: Built entirely on parameterized PDO statements.

Cross-Site Request Forgery (CSRF) Tokens: Protects sensitive forms (like deleting or editing data) from malicious attacks.

Anti-Piracy License System: Includes an embedded hardware key configuration check inside db.php to completely lock out unauthorized software redistribution.

Clean Source Code Base: Zero bulky frameworks or complicated third-party dependencies, allowing for quick adjustments on standard cPanel configurations.
